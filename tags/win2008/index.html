Windows Server 2008启用无线局域网支持

公司的本装上Windows server 2008后无线网就不能用了，这对移动办公来说及其不方便，在网上搜了解决方法，记录如下： 1 打开“服务器管理器”，如下 2 选择“功能”，点击“添加功能”，在弹出的向导列表中勾选“无线 LAN 服务”进行安装。 3 安装成功后，无线网就可以正常使用了。
