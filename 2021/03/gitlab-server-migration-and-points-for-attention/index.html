有两个地方需要注意：

1、在外网服务器上需要使用 nginx 进行代理访问，配置如下：

  • listen：外网访问的端口
  • server_name：外网服务器 ip
  • proxy_pass：内网 GitLab 的访问地址

2、默认情况下，内网部署的 GitLab 初始化的访问地址也是内网的地址，在界面中看到仓库地址也是内网地址，如下图：
