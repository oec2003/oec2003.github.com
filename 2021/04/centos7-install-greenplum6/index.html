CentOS 7 安装 Greenplum6 （附 dotNET Core 示例）

Mysql 在面对大数据量的时候，还是表现有些吃力，所以产品中需要扩展能支持海量数据的数据库，这里选择的数据库为 Greenplum6 ，Greenplum 底层使用的是开源数据库 PostgreSQL 。本文会介绍怎样在 CentOS 7 中安装 Greenplum6，并使用 dotNET Core 程序进行连接访问。

环境

  • CentOS：7.6
  • Greenplum：6.15
  • dotNET Core：3.1
  • NpgSql：5.0.4
