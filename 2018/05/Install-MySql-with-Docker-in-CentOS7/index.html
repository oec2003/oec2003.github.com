总结

本文介绍的方法虽然最终可以连接成功，但MySql的配置文件和数据都在容器内，如果由于配置原因导致容器无法启动，数据内容将会丢失，所以更好的做法是将配置文件和数据存储挂接到宿主机中，下一篇讲介绍怎样在MySql的容器中讲配置文件和数据目录挂接到宿主机中。
